# 📊 Telegram 考勤机器人

一个功能完善的 Telegram 考勤机器人，支持上班打卡、休息管理、工作时长统计等功能。

## ✨ 主要功能

### 📅 核心功能
- **上班打卡** (`/startwork`) - 开始记录工作时间
- **休息管理** (`/break` / `/resume`) - 记录休息时间
- **状态查询** (`/status`) - 查看当前工作状态
- **数据统计** (`/stats`) - 查看今日工作统计
- **下班报告** (`/endwork`) - 生成每日工作总结

### 🎯 特色功能
- 📊 自动计算工作效率
- 💾 数据自动备份
- ⌚ 精确到秒的时间统计
- 📱 响应式键盘
- 🔄 实时状态更新
